Project information

  • Project date: 11 July, 2021
  • Project URL: Galaxy Shooter

Overview

Galaxy Shooter is a 2D game where the player has to shoot down asteroid and enemy objects using lasers and power ups such triple laser shots, shields.

Key Takeaways

  • I have mastered beginner C# concepts, like variables, "if" statements, and arrays.
  • I have implemented features to detect collisions, receive user input, and create player movements, collect and destroy game objects.
  • I also created simple power-ups including triple shots, laser beams, speed boosts, and shields.
  • I applied shaders that transform my game backgrounds.
  • I implemented sound effects, background music, and particle effects.
  • Through this game, I learnt how to navigate in the Unity Engine and discovered unique features like Asset Store.
